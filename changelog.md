# ucchip-gnss-top - Release Notes
======================================
## 1.1.12(2023.8.15)
* 功能优化
  + 更新解析库，处理部分 GSA 数据解析卫星信号失败
  + 优化 playback 数据读取和串口上报数据显示，过滤掉乱码数据
## 1.1.11(2023.7.12)
* 功能优化
  + 添加 ntrip 功能。通过 tcp 获取挂载点以及连接 Ntrip 服务并将获取到的数据通过串口写入板子
  + 优化 playback 数据读出，过滤掉乱码数据
## 1.1.10(2023.5.17)
* 功能优化
  + 修改主进程与渲染进程实时通讯方式，通过 webContents 方式从主进程向渲染进程实时发送数据
  + 修改数据解析后的数据格式，避免使用 json 序列化，优化程序性能
  + 调整渲染进程数据渲染，通过节流函数每两秒执行一次数据渲染
  + 调整历史日志保存长度，从之前 4 分钟左右清一次改为 30s清一次
  + 优化基础信息显示
## 1.1.9(2023.5.10)
* 功能优化
  + 处理读取串口数据时，拔掉串口后无法断开串口连接问题
  + 优化渲染进程数据处理
## 1.1.8(2023.4.15)
* 功能优化
  + 日志文件选择改为通过主进程调用系统文件选择，这样选择空文件夹也能拿到文件夹路径
  + 日志存储从渲染进程改到主进程中，文件写入方式也改为 writeStream 来实时写入日志以节约内存，提高性能
  + 修复程序内存溢出导致程序崩溃问题，优化渲染进程中的变量保存方式
  + 修复程序长时间运行卡顿问题，在主进程中读取串口数据后及时释放内存
## 1.1.1 (2022.8.24)
* 功能优化
  + 选择log文件存储路径时，只能选中存储路径的文件夹，但是对于.log文件不能进行命名，所以在选择路径时需添加对.log文件进行命名的功能；
  + 靶心图界面设定值按钮选中后设置真实的经纬度后，靶心图上不显示数据；
  + 对于靶心图显示的数据需添加清除功能；
  + 最大化串口信息框或地图界面框后用按钮Esc返回后,最大化界面按钮显示错误。
  + 修改因为屏幕分辨率问题导致靶心图设定值输入框不显示，修改为弹框形式
## 1.1.0 (2022.8.20)
* 功能优化
  + 靶心图功能缺失（已添加）；
  + 信号强度窗口显示问题：显示条会出现来回跳的情况（当某颗卫星由参与定位变成未参与定位时，会由排在左边区域切换到右边区域）（已优化）；
  + 串口信息窗口保存log不方便，需要提前创建一个文件后再选择它，希望做成软件窗口选择某个文件，当这个文件不存在时就新建（已添加）；
  + 串口信息窗口每次只保存了4k数据（已修复）。
